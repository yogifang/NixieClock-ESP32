# NixieClock-ESP32
Build a Clock with  Nixie Tubes 
1. MCU : ESP32-32D
2. System : Arduino 
3. Development tools : Visual Studio Code + PlatformIO + Kicad
4. Functions : Webpage Setting Wifi SSID , password & Time Zone , NTP client sync time once every day.
